Sobre fotos

Tenho gosto por fotografia de paisagens, talvez um pouco sobre o tempo, ver como a luz e os efeitos de clima se misturam às cores de cada elemento a minha volta.

Existem muitas teorias de como se tirar uma foto, mas não há um ponto específico para se tirar uma foto, apenas um ângulo ideal, uma sombra específica para determinado tipo de luz.

"A fotografia é a única coisa que controla o tempo, pois em apenas um pedaço de papel, é possível pará-lo".-Salvioni
